# Password Checker 🔒

Программа на **Go** для проверки сложности пароля. Она анализирует пароль, определяет уровень безопасности и сообщает, чего не хватает для повышения уровня.

---

## Возможности

- Проверка на наличие:
  - цифр
  - заглавных букв
  - строчных букв
  - спецсимволов
- Определение уровня пароля:
  - **СЛАБЫЙ**
  - **СРЕДНИЙ**
  - **СИЛЬНЫЙ**
- Вывод рекомендаций по улучшению пароля

---
